\# Spring Boot RestifyDB CRUD



This project demonstrates \*\*automated CRUD operations using RestifyDB\*\* in a \*\*Spring Boot\*\* application. RestifyDB automatically exposes REST APIs for database entities with minimal boilerplate code.



> ⚠️ \*\*XAMPP is required\*\* to run this project, as RestifyDB relies on a PHP-based backend provided through XAMPP.



---



\## Tech Stack



\* Spring Boot

\* RestifyDB

\* PostgreSQL

\* Gradle

\* XAMPP (Apache + PHP)



---



\## Prerequisites



Before running the project, make sure you have the following installed:



\* \*\*Java 17+\*\*

\* \*\*Gradle\*\*

\* \*\*PostgreSQL\*\*

\* \*\*XAMPP\*\* (Apache \& PHP enabled)



---



\## Features



\* Auto-generated REST APIs using RestifyDB

\* Zero-boilerplate CRUD operations

\* Spring Boot integration with external REST APIs

\* PostgreSQL database support
